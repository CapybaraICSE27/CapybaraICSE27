# Manual Review Results

This directory contains the human-filled manual review evidence for the
RQ1-RQ5 validation checkpoint. The package focuses on row-level auditor
evidence and the summary files derived from those rows.

The retained CSV files are compact auditor-facing views. They keep row identity,
category labels, review reasons, `manual_*` audit fields, and source-code
evidence or snippets. The memo and summary file report the adjudicated accuracy
results derived from those reviewed rows.

See `rq1-rq5-full-corpus-reviewed/` for the row-level files, index, and pruned
review bundle.
