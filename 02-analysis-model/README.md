# Analysis Model API Mappings

This directory contains the framework API mappings used to connect extracted
test-code operations to the paper categories for RQ1, RQ4, and RQ5.

## Contents

- `framework-api-mappings/framework_category_mapping.md`: scope notes and framework coverage.
- `framework-api-mappings/framework_api_mapping_schema.md`: shared field definitions for the mapping CSVs.
- `framework-api-mappings/rq1_setup_teardown_api_mapping.csv`: setup, teardown, and environment-control API mappings.
- `framework-api-mappings/rq4_ui_action_api_mapping.csv`: user-facing UI action API mappings.
- `framework-api-mappings/rq5_assertion_api_mapping.csv`: assertion and oracle API mappings.

The implementation that emits these feature records is included under
`05-implementation/`.
